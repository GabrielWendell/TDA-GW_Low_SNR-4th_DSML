1. Introduction & Context

  • This notebook documents the Topological Data Analysis (TDA) pipeline for detecting gravitational-wave-like signals under low signal-to-noise ratios (SNRs). Also, this notebook consolidates the diagnostic plots, statistical summaries, and interpretative notes that support the DSML 2025 poster figures.

Pipeline

Signal Generation $\implies$ Embedding $\implies$ Persistent Homology $\implies$ Vectorization (PI, PL, BC) $\implies$ Analysis

  • While PI and PL features tend to correlate (reflecting similar persistence energy distributions), BC features emphasize birthdeath count dynamics.

8. Conclusions & Next Steps

Summary

  • Persistence and vectorized descriptors (PI, PL, BC) are consistent and physically interpretable.
  • Topological features stabilize with increasing SNR.
  • Each vectorization encodes complementary structure: PI (spatial density), PL (functional shape), BC (temporal count evolution).
